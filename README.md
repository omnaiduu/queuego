# QueueGo Monorepo

A monorepo built with Bun, containing frontend and backend packages.

## Structure

- `packages/frontend` - Frontend application
- `packages/backend` - Backend application

## Getting Started

### Install dependencies

```bash
bun install
```

### Development

Run all packages in development mode:

```bash
bun run dev
```

### Build

Build all packages:

```bash
bun run build
```

## Workspaces

This project uses Bun workspaces to manage multiple packages. See [Bun Workspaces](https://bun.sh/docs/install/workspaces) for more information.
